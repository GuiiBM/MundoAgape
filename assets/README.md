## Instruções para adicionar os logos

Para adicionar as imagens dos logos no header, siga os passos:

1. **Coloque as imagens na pasta `assets/`:**
   - `logo-left.png` - Logo da esquerda
   - `logo-right.png` - Logo da direita

2. **Formatos suportados:**
   - PNG
   - JPG/JPEG
   - SVG
   - WebP

3. **Recomendações:**
   - Altura ideal: 60px (será redimensionada automaticamente)
   - Proporção: qualquer uma (mantém a proporção original)
   - Fundo: transparente (PNG) ou branco

4. **Exemplo de estrutura:**
   ```
   MundoAgape/
   ├── index.html
   ├── styles.css
   ├── app.js
   ├── data.js
   └── assets/
       ├── logo-left.png
       └── logo-right.png
   ```

As imagens serão exibidas automaticamente no header, uma de cada lado do título "Mundo Ágape".
